# 重构 ElegantLaTeX 模板

使用 dtx 和 ins 文件对模板进行重构，减少重复代码的维护。

备注：尽量与 ElegantBook 4.1 同步发布，最终采用单文件 dtx。

```shell
xelatex elegantlatex.dtx
```

source code of ElegantLaTeX templates（reconstruction）
